Get stoic quotes displayed within your WordPress admin dashboard.

== Description ==

Each time the WordPress admin dashboard section gets opened, a random stoic quote will be displayed within a widget.

== Screenshots ==

1. Example quote

== Installation ==

1. Install and activate the plugin
2. Go to the admin section: Dashboard > Home, click the "Screen Options" button, and activate the checkbox "Stoic Quotes"
